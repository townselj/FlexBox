What is a flexbox?

The Flexbox Layout (Flexible Box) module (a W3C Candidate Recommendation as of October 2017) aims at providing a more efficient way to lay out, align and distribute space among items in a container, even when their size is unknown and/or dynamic (thus the word “flex”).

How do I apply it?

Flex can significally improve a website's layout. For example:

Here's a few example divs. Te vertical alignment looks ugly as of now. However, with flexbox, we can change that. 

        
          div { display: flex; justify-content: center;}

By definiting the divs as a flexbox, we are able to to move our divs to a horizontal line. The order still looks off, so we can change that with flexbox. 

        div { display: flex; justify-content: space-evenly; flex-wrap: wrap;}
