What is a flexbox?

Flexbox is a property for the display option in css. It can be used in order to make websites more responsive and flexible (pun intended)

How do I apply it?

Flex can significally improve a website's layout. For example:

Here's a few example divs. The vertical alignment looks ugly as of now. However, with flexbox, we can change that. 

        
          div { display: flex; justify-content: center;}

By definiting the divs as a flexbox, we are able to to move our divs to a horizontal line. The order still looks off, so we can change that with flexbox. 

        div {display: flex; justify-content: space-evenly; flex-direction: row-reverse;}

Using our code, we can change the direction of our divs and even their spacing.
